# SNIPR — сервис сокращения ссылок

Полнофункциональный фронтенд-прототип сервиса сокращения ссылок на React + Vite + Tailwind CSS.

## Что внутри

- **Лендинг** с описанием возможностей
- **Регистрация/авторизация** с поддержкой реферальных кодов через URL (`?ref=xxx`)
- **Дашборд** с 4 разделами:
  - **Мои ссылки** — создание (с кастомным алиасом), копирование, удаление
  - **Аналитика** — графики переходов, география, устройства, топ ссылок
  - **Кошелёк** — баланс с разбивкой (реклама + реферальные), вывод средств
  - **Рефералы** — ссылка для приглашения, код, список приглашённых, 10% комиссия
- **Редирект коротких ссылок** на `/r/:code` — с заставкой и регистрацией клика

## Стек

- **React 18** + React Router
- **Vite** — сборка
- **Tailwind CSS** — стили
- **Lucide React** — иконки
- **localStorage** — хранение данных (на этом этапе)

## Запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev
```

Откроется на `http://localhost:5173`.

## Сборка для продакшена

```bash
npm run build      # сборка в /dist
npm run preview    # локальный просмотр сборки
```

## Что нужно для перехода в продакшен

Этот проект — фронтенд-прототип. Все данные хранятся в `localStorage` браузера.
Чтобы превратить его в боевой сервис, нужно:

1. **Бэкенд** (Node.js + Express / FastAPI / Go и т.д.) с эндпоинтами:
   - `POST /api/auth/register` и `POST /api/auth/login` — с настоящим хешированием паролей (bcrypt) и JWT-токенами
   - `GET/POST/DELETE /api/links` — CRUD операций над ссылками
   - `GET /:shortCode` — серверный 302-редирект (это критично: сейчас редирект на клиенте, что медленно и не работает без JS)
   - `GET /api/analytics` — агрегация данных
   - `POST /api/withdrawals` — заявки на вывод средств

2. **База данных**: PostgreSQL для пользователей/ссылок/выводов, ClickHouse или TimescaleDB для аналитики кликов (там много данных)

3. **Защита**: rate limiting, CAPTCHA на регистрации, проверка ссылок через Google Safe Browsing API

4. **Интеграции**:
   - Рекламная сеть для монетизации (например, AdsTerra, PropellerAds)
   - Платёжный провайдер для выводов (ЮKassa, СБП, USDT)
   - GeoIP для определения страны (MaxMind)

5. **Замена в коде**:
   - `src/lib/storage.js` → fetch-обёртка к API
   - `src/lib/utils.js` → удалить `simulateClick` и `hashPassword`
   - В `LinksTab.jsx` убрать кнопку «+1 клик» (она была демо)
